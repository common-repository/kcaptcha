== Description ==

Kcaptcha plugin is the perfect security plugin for your wordpress website forms that protects your website from spam bots. Kcaptcha allows you to implement security captcha anywhere in website’s form. It is very easy for humans and hard for robots. Kcaptcha is designed to recognise the difference between a bot and human and sometimes even malicious software to provide the security to the system. This captcha can be used for login, registration,password recovery, comments and contact forms.If you want to prevent the entry of bots into your website, then you must start using Kcaptcha as it is the smartest way to stop it.

= Features =

Add captcha to: 

Registration form

Login form

Reset password form

Comments form

Easy for humans and hard for robots 


== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
In setting of this plugin you can find checkboxes. Save this entry.

Add a shortcode [addKcaptcha] below a form. 

For validation you have to call validateKCaptcha() function.
